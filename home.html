This site helps you pick color schemes defined by different rules which you choose. Just pick a base color and this site will generate you 4 more colors coresponding to the ruleset you have chosen. You can also lock colors you want to keep (although locking means the color wont affect the others colors in any way). There is also an option to apply the current schem to the site so you can see it all in action
